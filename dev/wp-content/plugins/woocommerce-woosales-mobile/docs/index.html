Description

WooSales Mobile is a very useful plugin that extends your WooCommerce installation with a very effective mobile reporting system.

What functionalities you'll have:

  • live sales reporting in your pocket!

Installation

  1. Copy 'woocommerce-woosales-mobile' folder into your WordPress plugin's folder and activate WooSales Mobile. WooSales Mobile requires WooCommerce plugin.

Usage
After you activate WooSales Mobile please navigate to WooCommerce -> WooSales Mobile page. There you have your mobile API key and a link to WooSales Mobile iOS and Android apps. Install the application and login with your website URL and API key provided.
(you can change the default API key by setting your own)
